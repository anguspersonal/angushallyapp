# Vision

**angushallyapp** is a personal web platform that serves three concurrent purposes:

1. **Portfolio** — showcase Angus Hally's work, skills, and thinking.
2. **Personal tooling** — small, useful tools for Angus, friends, and family (habit tracking, bookmarks, food-safety lookup, etc.).
3. **Playground** — a space to explore new technologies, patterns, and product ideas.

## Audiences

The site is built to serve multiple audiences at once; the homepage leads with identity and lets visitors self-select:

- Curious investors assessing credibility and engagement in HeyLina.
- Future collaborators, advisors, and potential hires.
- Blog readers (and, more durably, a record for Angus's children).
- Opportunistic service enquiries (web dev, consulting, maths tutoring) via hidden landing pages — not a storefront.

## Strategic goals

1. **Content management** — flexible, modern blog; portfolio pages; rich media.
2. **Personal tooling** — habit tracking (Strava, RescueTime), bookmark intelligence, UK food-safety lookup, time tracking.
3. **User experience** — cohesive identity-first interface, responsive, robust auth.

## Long-term direction

- Expand personal-productivity tools.
- Explore ML for habit insights.
- Consider a mobile companion.
- Expose select capabilities via API for third-party integration.
