# Wow Factor Improvement Plan

**Date:** 2026-04-10
**Purpose:** Ideas for making the webapp impressive as a demo portfolio site.

---

## Priority Tiers

### High Impact, Moderate Effort
1. **Scroll-driven landing page narrative** — Replace the static hero with a scroll-driven experience telling the journey (TeachFirst → Accenture → Anmut → HeyLina) visually as the user scrolls. Each section reveals with purpose.
2. **Smooth page transitions** — `AnimatePresence` + layout animations between routes. Hard cuts → cross-fades.
3. **AI chat widget** — Small chat bubble letting visitors ask about Angus. Powered by OpenAI with CV/about/blog content as context. Demonstrates AI chops as COO of HeyLina.

### High Impact, Lower Effort
4. **Typography upgrade** — Display/heading font (Space Grotesk, Clash Display, Satoshi) paired with Ubuntu body text.
5. **Cursor/hover micro-interactions** — Magnetic buttons, custom cursor on interactive elements, smooth hover transitions revealing more info.
6. **Dark mode with beautiful toggle** — Already have dark color (#141517). Add a satisfying toggle animation.

### Medium Impact, Creative Differentiators
7. **Command palette** — `Cmd+K` to search/navigate the site. Libraries like `cmdk`. Unexpected on personal site.
8. **Live dashboard element** — Strava/GitHub data on landing page as ambient visualization.
9. **Project case studies** — Transform 2-3 projects from grid cards into rich narrative pages showing process.

---

## Audience Considerations

- **Technical/startup people:** Prioritize AI chat (#3), scroll narrative (#1), case studies (#9)
- **Broader audience (investors, clients):** Prioritize page transitions (#2), typography (#4), landing redesign (#1), AI chat (#3)
