A personal habit-logging tool. The page used to track alcohol, exercise,
and other intentional habits over time with per-user data isolation and
charting.

It's now archived: the backend login system and database that powered it
are no longer maintained. The /projects/habit URL still resolves so the
page can sit in the timeline of past work, but the tracker itself does
not work.
