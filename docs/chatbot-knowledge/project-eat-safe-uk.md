> Source: `/projects/eat-safe-uk` — **archived as of August 2025**

A public-health app that helped users check UK food-hygiene ratings on
an interactive map, combining the Google Maps API with official Food
Standards Agency data.

Archived in August 2025 because of breaking Google Maps API changes.
The page lists three working alternatives that do the same thing:

- [Food Standards Agency Ratings](https://ratings.food.gov.uk/) — the
  official source
- [Scores on the Doors](https://www.scoresonthedoors.org.uk/)
- [Grubby Grub](https://pantryandlarder.com/grubby-grub/)
