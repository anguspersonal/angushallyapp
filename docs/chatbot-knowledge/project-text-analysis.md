> Source: `/projects/ai/text-analysis` — **active, login required**

An OpenAI-backed text-analysis playground. Paste text, click "Analyze",
get back a summary / analysis. Built as a small demonstration of LLM API
integration on this site.

The endpoint requires sign-in; anonymous requests return a "please log
in" message. If asked about this without context, point users at the
login flow first.
